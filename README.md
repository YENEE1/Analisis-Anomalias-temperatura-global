# 🌍 Global Temperature Anomaly Analysis (1850–2021)

## 📌 Project Overview

This project analyzes global monthly temperature anomalies from 1850 to 2021 using Python. The objective is to identify long-term trends, seasonal patterns, and historical changes through data cleaning, exploratory data analysis (EDA), and data visualization.

---

## 🎯 Objectives

- Clean and validate the dataset.
- Analyze long-term temperature trends.
- Compare anomalies by year, decade, month, and quarter.
- Identify the warmest and coldest periods.
- Visualize the evolution of global temperature anomalies.

---

## 🗂 Dataset

Source:

- Berkeley Earth
- NOAA Physical Sciences Laboratory (PSL)

The dataset contains monthly global temperature anomalies beginning in January 1850.

---

## 📊 Analysis Performed

### Data Cleaning

- Data type verification
- Date conversion
- Missing value verification
- Duplicate verification
- Outlier detection

### Exploratory Data Analysis (EDA)

- Descriptive statistics
- Annual average temperature anomalies
- Decadal analysis
- Monthly analysis
- Quarterly analysis
- Long-term trend visualization

---

## 📈 Key Findings

- Global temperature anomalies show a clear upward trend since 1850.
- The warming trend becomes more pronounced after the 1980s.
- The most recent decades present the highest anomaly values.
- Although annual fluctuations exist, the long-term trend remains positive.
