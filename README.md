# Analisis-Anomalias-temperatura-global
# Análisis de anomalías de temperatura global (1850–2021)

## Objetivo

Analizar la evolución de las anomalías de temperatura global mediante técnicas de limpieza, análisis exploratorio y visualización de datos con Python.

## Herramientas

- Python
- Pandas
- Matplotlib
- NumPy
- Google Colab

## Fuente de datos

Berkeley Earth / NOAA PSL

## Principales hallazgos

- Tendencia creciente de las anomalías desde 1850.
- Incremento acelerado a partir de la década de 1980.
- Las anomalías recientes permanecen en niveles históricamente elevados.
